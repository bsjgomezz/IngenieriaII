# 📘 Ingeniería II – Desarrollo Web

Repositorio correspondiente a la materia **Ingeniería II** de 2° año de la  
**Tecnicatura Superior en Desarrollo de Software**  
📍 Instituto Superior de Profesorado N° 20  

---

## 📖 Descripción

Este repositorio contiene los trabajos prácticos, ejercicios y proyectos desarrollados durante la cursada de la materia *Ingeniería II*, enfocados en el desarrollo de aplicaciones web utilizando tecnologías del lado del cliente.

El objetivo principal es adquirir y fortalecer conocimientos en diseño, estructura y desarrollo de interfaces web dinámicas y responsivas.

---

## 🚀 Tecnologías utilizadas

Durante la cursada se trabajará con las siguientes tecnologías:

- 🧱 **HTML5** → Estructura de las páginas web  
- 🎨 **CSS3** → Estilos y diseño visual  
- ⚙️ **JavaScript** → Interactividad y lógica del cliente  
- 🅱️ **Bootstrap** → Framework para diseño responsivo y componentes UI
